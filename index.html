Work

Heima (A Cozy Front-End Library)

A front-end library that is meant to invoke a homey feeling, the feeling I get when writing code with a hot beverage. Once I started really getting into Sass and front-end development, I noticed I would often go back to older projects and resuse components I made before. My younger brother observed the same and together we decided to start building an open source front end library together. We've started off with smaller components, like simple buttons and animations. We have large plans for Heima and welcome anyone interested to contribute. Heima means "at home" in Icelandic, and that's the feeling we hope it instills in the folks that use it.

HTML Sass/CSS

Math Problem Generator

Angular 2 application for grade school math teachers that allows the user to enter an equation, set parameters for variables (decimal point, minimum value, maximum value), choose the variable to sovle for, number of problems desired and then displays a table of values for each variable. In this project my small team identified and addressed existing tech debt as well as refactored the existing code base using higher order functions and reduced total lines of code by 60%. Additionally, we throughly documented the code base for those that would follow, implemented bug-fixes, and added on new features, including accounting for decimal variables within the equations.

Angular CLI JavaScript TypeScript Jasmine Bower NPM Node JS Karma Guppy Bootstrap 4 Nerdamer Lodash

Eat Up

Angular 2 Web Application that allows users to create an account and search for restaurants by cuisine type within a specified city. Users can sign in or create a new account (including google sign in). Once singed in you users can navigate to their dashboard, search for resaturants, or log out. While on the search page, users can search by food type and city. The search results will yield a map of the city and a list of restaurants. By clicking the "love it" button users will add the restaurant to their dashboard. From the dashboard, users can upload a profile photo and edit their list of loved restaurants. Personally responsible for Google Maps Integration and Google Places API calls as well as Map/User dashboard views and storage of user data in firebase.

Angular 2 HTML CSS Javascript/jQuery NPM Bower Bootstrap Firebase

Timber Tigers Hiking Club

Angular 2 Web Application to keep track of the members of Timber Tigers Hiking Club (a hiking club I started with some friends). Users can view a list of current members and click on them to navigate to their profile page to view more information about that member. Administrators can add new members, edit exisiting members and remove members that are no longer involved with the club.

Angular 2 HTML CSS Node NPM Bower Bootstrap Firebase

Doctor Lookup

JavaScript Web Application that utilizes the Better Doctor API. This application allows the user to enter a medical issue or name and then returns a list of doctors with their first and last name, address, phone number, website and whether or not ther are accepting new patients. If the API call returns no results, it informs the users that no results were found for their query.

JavaScript HTML CSS Node Gulp Jasmine Karma BetterDoctor API
